Использование gpart на сервере FreeBSD (Sparc64 Sun v215)

Description

Собственно всё как в мануалах, для разметки используем VTOC8.

gpart create -s VTOC8 -f x /dev/xxx
gpart commit

У меня получилось вот так

#gpart show
=>        0  143203410  da0  VTOC8  (68G)
